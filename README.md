# 被験者募集サイト（実験研究プロジェクト）

本プロジェクトは、大学・研究機関における心理学・睡眠・行動科学などの実験に参加してくださる **被験者募集用ウェブサイト** です。  
参加希望者が条件を確認し、応募フォームから登録できるようになっています。

---

##  サイトの概要

- 実験の目的や概要をわかりやすく掲載
- 募集対象（例：18〜35歳の健康な成人など）
- 実験の所要時間・報酬・実施場所を明記
- Google フォームによる応募受付を実装
- スマートフォン対応デザイン（レスポンシブ）

---

##  使用技術

- **HTML / CSS / JavaScript**  
- **Google スプレッドシート**（データ連携）  
- **GitHub Pages**（公開用ホスティング）
